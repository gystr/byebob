# byebob

A CLI tool for automating attendance tasks in hiBob.

## Installation

```bash
npm install -g byebob
```

## Usage


#### Configure credentials:
```bash
byebob configure
```

#### Fill attendance:
```bash
byebob fill
```

#### Submit attendance report:
```bash
byebob submit
```
